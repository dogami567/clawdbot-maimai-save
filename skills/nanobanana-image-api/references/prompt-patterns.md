# Prompt Patterns

Use these templates when the user gives weak or ambiguous prompt input.

## Text To Image

Template:

`[subject]. Style: [style]. Composition: [framing]. Lighting: [lighting]. Quality: [quality]. Avoid: [negative].`

Example:

`A banana-themed sneaker product poster. Style: modern luxury advertising. Composition: centered hero shot, clean background. Lighting: softbox studio light with subtle rim light. Quality: high detail, crisp texture, 4k-ready render. Avoid: watermark, text overlays, blur, low contrast.`

## Image To Image

Template:

`Transform the reference image into [target outcome] while preserving [elements to keep]. Style: [style]. Add [new elements]. Keep [constraints].`

Example:

`Transform the reference image into a premium ecommerce hero banner while preserving product shape and logo placement. Style: clean commercial photography. Add soft reflection and minimal props. Keep realistic materials and true product color.`

## Character / Mascot

Template:

`Create [character] in [scene]. Style: [visual style]. Pose: [pose]. Camera: [camera]. Mood: [mood].`

Example:

`Create a friendly banana mascot in a rainy neon street. Style: cinematic 3D illustration. Pose: walking with umbrella. Camera: low-angle medium shot. Mood: playful but cool.`

## Fashion / Product Mockup

Template:

`Generate a [product type] mockup using [material/features]. Background: [background]. Lighting: [lighting]. Output intent: [ad/social/poster].`

Example:

`Generate a streetwear hoodie mockup using thick cotton and embossed banana icon embroidery. Background: muted concrete wall with gradient wash. Lighting: directional soft light, realistic shadows. Output intent: social ad cover image.`

## Iteration Delta Template

Use this template when AI updates prompt after a failed round:

`Keep [what worked]. Fix [top 2 problems]. Increase [detail/contrast/composition instruction]. Avoid [new artifacts].`

Example:

`Keep the premium sneaker texture and neon mood. Fix weak product focus and cluttered background. Increase center framing and edge sharpness. Avoid watermark, extra text, and distorted logo.`
